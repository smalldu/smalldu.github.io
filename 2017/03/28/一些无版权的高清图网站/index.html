无版权的高清图网站

  • 60 余家免费正版图片网站，应该是目前最全的的集合了
  • Gratisography
  • PicJumbo
  • Lifeofpix 图片多为欧洲景观
  • Imcreator 该站提供搜索和分类目录，有人物、自然、艺术、生活、图标等分类。确实不错，可惜有的分类很少图。
  • Freeimages 图片多为欧洲景观
  • Deathtothestockphoto 用户需要邮箱订阅后才能获取该网站上的图片，然后该站每月会向你发送免费的图片素材
  • Pixabay
  • Publicdomainarchive
  • snapographic
